# 🔑 Secure Password Generator (CLI)

An interactive, user-friendly, and secure Command-Line Interface (CLI) Password Generator built with Python. This project allows users to generate customizable, strong, and highly secure passwords by defining their preferred length and character sets[cite: 1].

Developed as part of the **Python Development Internship** at **Synent Technologies**[cite: 1].

---

## 🚀 Features
- **Customizable Length:** Users can specify any password length (minimum 6 characters for security)[cite: 1].
- **Selective Character Pools:** Toggle uppercase letters, numbers, and special characters on or off[cite: 1].
- **Input Validation:** Prevents crashes from invalid inputs (e.g., non-integer entries or passwords shorter than the secure minimum limit)[cite: 1].
- **Clean CLI Experience:** Modern and visually neat terminal styling[cite: 1].

## 🛠️ Technology Used
- **Language:** Python 3.x[cite: 1]
- **Core Modules:** `random`, `string` (Built-in)[cite: 1]
